No nosso projeto foram usados varios componentes na composição, seguem alguns dos mais vitais na composição:

Arduíno uno R3

Usado para a programação do carrinho.

Motores DC 3

Usados para mover o carrinho.

Mini Protoboard

Usado para a conexão dos componentes ao código do Arduíno.

Sensor Ultrassônico De Distancia Hc-sr04

Usado para evitar obstáculos.

01 - Módulo Bluetooth Rs232 Hc-05

Usado no modo bluetooth do carrinho.

Sensores infravermelho TCRT5.000

Usado para detectar a linha.
